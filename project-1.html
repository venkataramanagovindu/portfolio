Medical Consultancy System

A web-based platform that streamlines appointment scheduling, patient record management, and digital prescriptions for healthcare professionals. Built with .Net Core MAVC, Azure SQL and Azure, ensuring secure and efficient medical consultations.

Live Link

Project Overview

Project Overview The Medical Consultancy System is designed to streamline medical consultations, appointments, and patient management for healthcare professionals. The system provides an intuitive interface for both doctors and patients, ensuring efficient scheduling, diagnosis tracking, and data management.

Features

Appointment Scheduling: Enables patients to book, reschedule, and cancel appointments online. Patient Records Management: Stores and retrieves patient medical history, prescriptions, and reports. Doctorâ€™s Dashboard: Provides doctors with an overview of upcoming appointments, patient records, and medical notes. Secure Login & Authentication: Ensures user data security through role-based access controls. Prescription Management: Allows doctors to prescribe medicines digitally and share them with patients. Notifications & Reminders: Sends automated email or SMS notifications for appointments and prescription refills.

Technologies Used

Frontend: ASP.NET Core MVC for dynamic UI and responsive web application. Backend: .NET Core MVC for handling business logic and API interactions. Database: Azure SQL Database with Code-First Approach for database modeling and migrations. Authentication & Security: Implemented JWT-based authentication and role-based access control (RBAC). Deployment & Hosting: Hosted on Azure App Services with Azure SQL as the backend database. DevOps & CI/CD: Implemented CI/CD pipeline using Azure DevOps for seamless deployment. Logging & Monitoring: Integrated Application Insights and Azure Monitor for real-time tracking. Version Control: Managed source code using Git & GitHub.

Role & Contributions

Developed a scalable .NET Core MVC web application with clean architecture principles. Implemented the Code-First Approach to design database models and manage migrations. Developed RESTful APIs to handle data transactions between frontend and backend. Optimized Azure SQL queries to enhance performance and reduce latency. Deployed the application on Azure App Services and configured Azure Blob Storage for document storage. Configured logging and monitoring using Azure Application Insights. Implemented Azure Active Directory (AAD) authentication for secure user access.

Outcome & Impact

Successfully deployed a cloud-based application with high availability and scalability on Azure. Achieved seamless DevOps automation with CI/CD integration via Azure DevOps Pipelines.
